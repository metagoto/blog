Formating help

Text in your comment is automatically escaped. Paragraphes will be enclosed with <p> elements.

You can submit snippet codes by surrounding them with --- (three dashes) at the beginning of lines.

When in doubt, use brute force preview your comment!

Example:

Dummy comment blah
---
function foo(bar) {
  // working so far
}
---
Another para

Result:

Dummy comment blah

function foo(bar) {
  // working so far
}

Another para
